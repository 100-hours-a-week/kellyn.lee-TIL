### [대상과 구조]

- 수학의(적) 대상 : 집합과 함수
- **집합, collection** : 수학적 대상(수, 함수, 집합)을 포함하는 모임
    
    $$
    \begin{aligned}
    \mathbb{N} &= \{1, 2, 3, \dots\}\\
    \mathbb{Z} &= \{\dots, -2, -1, 0, 1, 2, \dots\}\\
    \mathbb{R} &= (-\infty, \infty)
    \end{aligned}
    $$
    
- **멱집합, family collection** : 모든 부분집합을 모은 집합
    
- ***벡터 공간 = 대수(적) 구조*** : 집합에 연산을 추가
    
    <aside>
    💡 어떤 연산을 추가한 것 일까? **벡터 ‘합’ & 스칼라 ‘곱’** (연산 2개)
    </aside>
    
    - 벡터 공간은 8개의 공리를 만족해야함.
        
        V = (벡터의 집합, +, *)
        
        *→ V : 벡터들의 집합 / + : 벡터 덧셈 연산 / * : 스칼라 곱 연산*
        
- **구조 = 튜플 (집합, 연산, 성질) = 집합과 특징들의 결합**

### [대수 구조]

- **대수 구조** : 집합 + 연산 + 공리(집합과 연산이 만족시켜야하는 조건)
    - **집합, set** : 원소들의 모임
    - **연산, operation** : 두 원소를 결합하여 새로운 원소를 생성하는 함수
    - **공리, Axioms** : 연산이 반드시 만족해야 하는 **조건(명제)**

- **마그마, magma**
    - (S, *) (이항연산) : S x S → S 함수 = 어떤 연산이 어떤 집합에 닫혀 있음.
    - 집합 S의 원소끼리 연산하면 다시 S의 원소가 되어야 함.



### 군, Group

- 마그마에서 몇 가지 추가 조건을 만족하는 구조 / **(G, *)**
    
    [군의 공리]
    
    1. ***결합법칙, Associativity***
        1. 연산의 괄호를 변경해도 결과가 동일해야함. 
        2. 연산의 순서를 자유롭게 할 수 있음.
            
            (a*b)*c = a*(b*c)
            
            → 이를 만족할 경우 **반군(Semigroup)**이라고 부름
            
    2. ***항등원, Identity Element*** 
        1. 어떤 원소와 연산을 해도 변하지 않는 원소(항등원 e)가 있어야 함.
            
            a * e = e * a = a
            
            → 이를 만족할 경우 **모노이드(Monoid)**라고 부름
            
    3. ***역원, Inverse Element*** 
        1. 모든 원소 a에 대해 어떤 원소 x가 존재하여 아래 수식이 성립해야함.
            
            a * x = x * a = e


     
            a + x = 0, x = -a
            
    
    <aside>
    💡 교환법칙도 만족할 경우 아벨 군(Abelian Group) → a * b = b * a
    *% 아벨군 = 가환군 %*
    
    </aside>
    


### 환, Ring

- 두 개의 연산(덧셈과 곱셈)을 가지는 대수 구조 / **(R, +, *)**
    
    [환의 공리]
    
    1. ***덧셈(+)은 아벨 군*** 형성
        1. 결합법칙, 항등원(0), 역원(-a), 교환법칙(가환성) 만족
    2. ***곱셈(x)은 모노이드*** 형성
        1. 결합법칙, 항등원 존재 (하지만 역원은 필요 없음)
    3. ***분배법칙, distributive law***
        1. 곱셈이 덧셈에 대해 분배되어야 함.
            
            a*(b+c)=a*b+a*c


       
            (a+b)*c=a*c+b*c
            
    
    <aside>
    💡 환 (ring) = 정수 집합 (Z, +, *) ≠ 체 (field)
    
    </aside>
    




### 체, Field

- 환의 확장판, 곱셈 연산에 대한 역원이 존재해야함 / **(F, +, *)**
    
    [체의 공리]
    
    1. ***덧셈 연산은 아벨군***
    2. ***곱셈 연산도 아벨군*** (0을 제외한 원소에 대해 역원이 존재해야 함)
       
        



### 벡터공간의 공리

- 벡터 덧셈과 스칼라 곱 : **10개의 공리**를 만족해야함.
    1. 벡터 덧셈에 대한 아벨군 구조 (8개 공리)
        1. 결합법칙, 교환법칙, 항등원, 역원 존재 등
    2. 스칼라 곱에 대한 두 개의 공리 (2개)
        
        
        (ab)*v &= a*(b*v)
 

  
        1*v &= v
        
        
    4. 분배법칙 (2개)
        
        
        a*(u+v) &= a*u + a*v
 
  
        (a+b)*v &= a*v + b*v
        
        
        *→ 벡터공간을 일반화하면 모듈(Module)*

### [공간]

| 공간의 종류 | 구조 | 의미 |
| --- | --- | --- |
| 벡터 공간, Vector Space | 덧셈, 스칼라 곱 | 벡터끼리의 연산이 가능 |
| 내적 공간, Inner Product Space | 내적(*) 추가 | 벡터 사이의 유사성 측정 가능 |
| 놈 공간, Normed Space | 놈(벡터의 크기) 추가 | 벡터의 크기 정의 가능 |
| 거리 공간, Metric Space | 거리 함수 추가 | 두 벡터 간 거리 정의 기능 |
| 위상 공간, Topological Space | 열린 집합 추가 | 연속성과 위상적 성질 정의 가능 |

**※ 내적 공간 < 놈 공간 < 거리 공간 < 위상 공간 ※**



### 놈 공간, Normed Space

- 벡터 공간에서 놈(norm, 벡터의 크기)을 정의한 공간 / **(V, ||*||)**
    - **놈** : 벡터의 크기를 측정하는 함수, 내적으로부터 유도될 수 있음
       *→ 놈이 정의되면, 벡터의 길이(크기)를 계산할 수 있음*


    

### 거리 공간, Metric Space

- 놈이 정의되면, 이를 이용하여 두 벡터 사이의 거리를 정의할 수 있음 / **(V,d)**
    - **거리 함수 d(x,y)**는 놈을 이용하여 정의
        
        d(x,y)=∥x−y∥
    
    *→ 놈이 존재하면 거리도 정의 가능 → 거리 공간 만들 수 있음.*
    




### 위상 공간, Topological Space

- 거리가 정의되면, 열린 집합(Open Set) 정의 가능 → 위상 공간 형성 / **(V, τ)**
    - τ : 열린 집합(연속성)의 모음

### [차원과 기저]

- 벡터 공간의 차원은 그 공간의 기저(Basis)의 원소 개수로 정의
    - **기저, Basis** : 벡터 공간에서 선형 독립, 공간을 생성(Span)하는 벡터들의 집합
        
        β={(1,0),(0,1)} in R^2
        
    - **차원, Dimension, dim V** : 기저의 원소 개수

**※ 넘파이 차원 = 수학의 텐서의 차수, 랭크 ※**

- 넘파이 `ndim` = 배열의 차원
- 행렬의 `Rank` = 선형 독립한 열의 개수
